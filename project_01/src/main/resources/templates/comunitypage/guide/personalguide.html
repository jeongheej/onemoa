개인회원 이용안내



1.

병원예약에서 원하시는 병원을 검색후 상세보기를 선택합니다.

2.

병원을 확인후 예약하기 선택

3.

개인 정보를 확인 후 원하시는 진료날짜 시간 진료내용 등을 입력 후 예약하기 클릭

4.

예약정보를 확인은 마이페이지 > 예약조회 및 취소에서 확인

5.

병원명을 선택 후 자세한 예약확인 가능, 취소시에는 체크박스를 선택후 예약취소 클릭

6.

예약내역 변경을 원할시 수정 버튼을 클릭 후 예약변경 가능
